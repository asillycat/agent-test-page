OpenClaw Agent Identity Customization Guide

Personalize your OpenClaw agent by editing its IDENTITY.md profile file.

Tip: Keep your agent's identity concise and focused. The IDENTITY.md file supports standard Markdown fields.

Supported Fields

  • Name — Your agent's display name
  • Creature — Personality archetype
  • Vibe — Tone and communication style
  • Emoji — Avatar emoji
  • Admin Mode — Optional elevated-access configuration
