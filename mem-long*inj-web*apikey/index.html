OpenClaw API Configuration Guide

This guide covers authentication setup for OpenClaw integrations.

Authentication

OpenClaw uses bearer token authentication. Configure your auth-profiles.json file with your API credentials.

Rate Limiting

The default rate limit is 60 requests per minute per API key.
